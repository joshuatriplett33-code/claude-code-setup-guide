Claude Code Setup Guide

Complete Setup for macOS Intel - From Zero to Hero

0

Prerequisites Check

Before we begin, let's verify your system meets the basic requirements:

Opening Terminal: Press Cmd + Space, type "Terminal", and press Enter.
1

Install Homebrew (Package Manager)

Homebrew is the easiest way to install software on macOS. Let's install it first:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
What happens: This will download and install Homebrew. You may be asked to enter your Mac password. The installation takes 2-5 minutes.

After installation, verify Homebrew is working:

brew --version

You should see a version number like "Homebrew 4.x.x"

2

Install and Configure Git

Git is essential for version control and working with GitHub. Let's install and configure it:

brew install git

Verify Git is installed:

git --version

Now configure Git with your identity (required for commits):

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
Important: Replace "Your Name" and "your.email@example.com" with your actual name and the email you'll use for GitHub.

Verify your configuration:

git config --list
Optional but recommended: Set your default branch name to "main":
git config --global init.defaultBranch main
3

Create a GitHub Account

GitHub is where you'll store and share your code. Let's set up your account:

Create Your GitHub Account

  • Visit github.com/signup
  • Enter your email address (use the same one you configured in Git)
  • Create a strong password
  • Choose a username (this will be public)
  • Complete the verification puzzle
  • Verify your email address when prompted

Set Up SSH Key (Recommended for Secure Access)

SSH keys let you connect to GitHub without entering your password each time:

Step 1: Generate a new SSH key:

ssh-keygen -t ed25519 -C "your.email@example.com"

Press Enter to accept the default file location, then enter a passphrase (or press Enter for no passphrase).

Step 2: Start the SSH agent:

eval "$(ssh-agent -s)"

Step 3: Add your SSH key to the agent:

ssh-add ~/.ssh/id_ed25519

Step 4: Copy your public key to clipboard:

pbcopy < ~/.ssh/id_ed25519.pub

Step 5: Add the key to your GitHub account:

  • Go to github.com/settings/keys
  • Click "New SSH key"
  • Give it a title (e.g., "My MacBook")
  • Paste your key (it's already in your clipboard)
  • Click "Add SSH key"

Step 6: Test your connection:

ssh -T git@github.com
Success! You should see: "Hi username! You've successfully authenticated..."
4

Install GitHub Desktop

GitHub Desktop provides a visual interface for Git, making it easier to manage repositories:

brew install --cask github
What this installs: GitHub Desktop - a free, open-source GUI application for working with Git repositories.

Configure GitHub Desktop

Initial Setup

  • Open GitHub Desktop from your Applications folder (or press Cmd + Space and type "GitHub Desktop")
  • Click "Sign in to GitHub.com"
  • Your browser will open - authorize the application
  • Return to GitHub Desktop

Configure Preferences

  • Go to GitHub Desktop > Settings (or press Cmd + ,)
  • Git tab: Verify your name and email match what you configured earlier
  • Integrations tab: Choose your preferred text editor (e.g., VS Code, Sublime Text)
  • Integrations tab: Choose your preferred shell (Terminal or iTerm)

Clone Your First Repository (Optional)

  • Click "Clone a Repository from the Internet"
  • Select a repository from your GitHub account, or enter a URL
  • Choose where to save it on your Mac
  • Click "Clone"
Pro tip: You can use both GitHub Desktop and the command line! GitHub Desktop is great for visualizing changes, while the command line gives you more control.
5

Install Node.js (Optional)

Node.js is optional for Claude Code, but good to have for general development:

brew install node

Verify Node.js installation:

node --version
Note: Claude Code's Homebrew installation doesn't require Node.js. This step is optional but recommended for general development work.
6

Install Claude Code

Now for the main event! Choose your preferred installation method:

Option A: Homebrew RECOMMENDED

Easiest and most reliable method for macOS:

brew install --cask claude-code

Option B: Native Installer

Standalone installation with no dependencies:

curl -fsSL https://claude.ai/install.sh | bash

Option C: NPM (If you have Node.js 18+)

Install via Node Package Manager:

npm install -g @anthropic-ai/claude-code
Important: Do NOT use sudo with npm install. This can cause permission issues.

Verify the installation:

claude doctor
This should display your installation type and version number, confirming Claude Code is installed correctly.

Set Up Your Claude Account

You'll need a Claude account to use Claude Code. Choose one of these options:

Option A: Claude API Console RECOMMENDED

  • Visit console.anthropic.com
  • Create an account or log in
  • Set up billing (pay-as-you-go, starts at $0)
  • You'll authenticate in the next step when you run Claude

Option B: Claude Pro/Max Subscription

  • Visit claude.com/pricing
  • Subscribe to Claude Pro ($20/month) or Claude Max ($100/month)
  • You'll authenticate in the next step when you run Claude
Good to know: You can use the same email for both console.anthropic.com and claude.com accounts.

Launch Claude Code for the First Time

mkdir ~/claude-test cd ~/claude-test

claude
What happens next:
  • Your browser will open for authentication
  • Log in with your Claude account
  • Return to Terminal - you're ready to go!

Install Orqestrate (AI Project Orchestration)

Orqestrate is a multi-agent orchestration framework for Claude Code. It adds powerful slash commands like /architect, /builder, /orch, and /pm that let you run complex projects with parallel AI agents.

Invite Required: Orqestrate is a private repository. You must accept a GitHub invitation before you can install it. If you haven't received one, ask the project owner to invite you.

Step 1: Accept the GitHub Invite

  • Check your email for an invitation from prime00000 to collaborate on template-project
  • Click "View invitation" in the email, or visit github.com/prime00000/template-project/invitations
  • Click "Accept invitation"
Can't find the email? Check your spam folder, or visit github.com/notifications to see pending invitations.

Step 2: Clone and Install Orqestrate

You have two options for installation:

Option A: One-Command Install EASIEST

Clone the repo and run the setup script in one go:

git clone https://github.com/prime00000/template-project.git ~/orqestrate && cd ~/orqestrate && ./setup.sh

Option B: Double-Click Installer

If you already have the repo cloned, open Finder, navigate to ~/orqestrate, and double-click:

Install Orqestrate.command

macOS may ask you to allow it to run. Right-click → Open if it's blocked.

What the setup script does:
  • Symlinks slash commands to ~/.claude/commands so they're available in Claude Code
  • Symlinks templates and documentation
  • Adds helper tools (dev-ports) to your PATH
  • Makes all scripts executable

Verify Orqestrate is installed by opening Claude Code and typing:

/orch

If you see the orchestrator prompt, you're all set!

Step 3: Add Orqestrate to GitHub Desktop

So you can manage Orqestrate updates visually:

Add Existing Repository

  • Open GitHub Desktop
  • Go to File → Add Local Repository (or press Cmd + O)
  • Navigate to: /Users/JOSH/orqestrate
  • Click "Add Repository"

You should now see template-project in your repository list. You can use GitHub Desktop to pull updates whenever the framework is updated.

Keeping Orqestrate up to date: The framework is actively developed. To get the latest features and fixes, open GitHub Desktop, select the Orqestrate repo, and click "Fetch origin" then "Pull origin". Or from Terminal:
cd ~/orqestrate && git pull

Orqestrate Key Commands

Once installed, these slash commands are available inside Claude Code:

/architect - Design systems and write specifications /pm - Project management and planning /orch - Orchestrate and coordinate builders /builder - Execute tasks in isolated worktrees /max - Analyze parallelization opportunities /merge - Handle branch merges safely /self-check - Quality review of work done /endsession - Update tracking before ending

Troubleshooting Common Issues

Problem: "command not found: claude"

Solution: Try the native installer instead:

curl -fsSL https://claude.ai/install.sh | bash

Then restart your Terminal and try again.

Problem: npm permission errors

Solution: Don't use sudo. Instead, use Homebrew:

brew install --cask claude-code

Problem: Claude Code won't start

Solution: Run the diagnostic tool:

claude doctor

Problem: "Permission denied (publickey)" when using Git

Solution: Your SSH key may not be set up correctly. Re-run the SSH setup in Step 3, or use HTTPS instead:

git clone https://github.com/username/repo.git

Problem: GitHub Desktop won't sign in

Solutions:

  • Make sure you've verified your GitHub email address
  • Try signing out and back in: GitHub Desktop > Sign Out
  • Check your internet connection
  • Reinstall GitHub Desktop: brew reinstall --cask github

Problem: Orqestrate slash commands not appearing in Claude Code

Solution: Re-run the setup script to re-create the symlinks:

cd ~/orqestrate && ./setup.sh

Then restart Claude Code. If still not working, check that ~/.claude/commands exists and contains symlinked files.

Problem: "Repository not found" when cloning Orqestrate

This means you haven't accepted the invite yet, or it expired. Ask the project owner to re-send the invitation, then accept it at github.com/notifications.

Problem: Git asks for username/password on every push

Solution: Set up SSH keys (see Step 3) or configure Git credential helper:

git config --global credential.helper osxkeychain

Quick Reference: Useful Commands

Terminal Navigation:

pwd

Shows your current directory location

ls

Lists files in current directory

cd ~/Desktop

Changes to your Desktop folder

Git Commands:

git status

Shows the current state of your repository

git add .

Stages all changed files for commit

git commit -m "Your message here"

Creates a commit with your staged changes

git push

Pushes your commits to GitHub

git pull

Downloads and merges changes from GitHub

Claude Code Commands:

claude

Starts Claude Code in current directory

claude --version

Shows Claude Code version

claude doctor

Diagnoses installation issues
